# ACE-Step Studio

[![Stars](https://img.shields.io/github/stars/timoncool/ACE-Step-Studio?style=social)](https://github.com/timoncool/ACE-Step-Studio/stargazers)
[![License](https://img.shields.io/github/license/timoncool/ACE-Step-Studio)](LICENSE)

**[English version](README.md)**

Портативная студия для генерации музыки с помощью ИИ. Создавайте песни с вокалом, текстами и музыкальными клипами — всё работает локально на вашей видеокарте. Без облака, без подписок, без интернета.

Построено на [ACE-Step 1.5 XL](https://github.com/ace-step/ACE-Step-1.5) (4B DiT модель) с полноценным веб-интерфейсом.

## Возможности

### Генерация музыки
- **Полноценные песни с вокалом** — до 8 минут, любой язык, любой жанр
- **Простой и продвинутый режимы** — опишите что хотите или настройте каждый параметр
- **3 XL модели** — XL Turbo (8 шагов, быстрая), XL SFT (50 шагов, максимальное качество), XL Turbo BF16 (компактная, 7.5 ГБ)
- **ИИ тексты и стиль** — LLM генерирует тексты и обогащает описания стиля
- **Горячая смена моделей** — переключение DiT/LM моделей без перезапуска
- **Пакетная генерация** — создавайте несколько вариаций за раз
- **ID3 теги** — MP3 файлы включают название, исполнителя, обложку, текст, BPM

### Видео Студия
- **Генератор музыкальных клипов** — визуализаторы в стиле NCS с 10 пресетами
- **Караоке субтитры** — синхронизированные LRC субтитры с 3 стилями (строки, бегущая строка, караоке заливка)
- **WYSIWYG редактор** — перетаскивание элементов, прокрутка для изменения размера
- **Соотношение сторон** — 16:9, 9:16 (Reels/TikTok), 1:1 (Instagram)
- **Эффекты** — тряска, глитч, VHS, сканлайны, bloom, плёночное зерно и другие
- **Серверный рендер** — нативный ffmpeg.exe с GPU ускорением NVENC

### Аудио инструменты
- **Аудио редактор** — обрезка, фейды, эффекты (AudioMass)
- **Разделение дорожек** — отделение вокала, барабанов, баса, прочего (Demucs)
- **Кавер и ремикс** — трансформация аудио с новыми стилями
- **Перерисовка аудио** — перегенерация отдельных секций

### Интерфейс
- **Один терминал** — один `run.bat`, Express управляет Python/Gradio
- **Портативность** — всё в одной папке, удалил = деинсталлировал
- **5 языков** — английский, русский, китайский, японский, корейский
- **Доступ по сети** — используйте с любого устройства в локальной сети
- **Мониторинг GPU** — живые статистики VRAM/RAM/CPU
- **Отслеживание времени генерации** — секундомер для каждого трека
- **Скачивание LRC** — экспорт синхронизированных текстов для Apple Music

## Быстрый старт

### 1. Клонировать

```bash
git clone https://github.com/timoncool/ACE-Step-Studio.git
cd ACE-Step-Studio
```

### 2. Установить

```
install.bat
```

Выберите тип видеокарты. Установит Python 3.12, PyTorch, Node.js 22 и все зависимости.

### 3. Запустить

```
run.bat
```

Один терминал — Express + Gradio пайплайн в одном окне. Браузер откроется автоматически когда модели загрузятся. Интерфейс на http://localhost:3001 (автоматически найдёт свободный порт).

Модели скачиваются автоматически при первом запуске (~7.5 ГБ для модели BF16 по умолчанию). Для предварительной загрузки используйте `download_model.bat`.

Для разработки с Vite HMR: `run-dev.bat` (3 терминала, фронтенд на http://localhost:3000).

## Архитектура

```
ACE-Step-Studio/
├── app/              # React + Express фронтенд и бэкенд
├── ACE-Step-1.5/     # Python ML пайплайн
├── python/           # Портативный Python 3.12 (создаётся install.bat)
├── node/             # Портативный Node.js 22 (создаётся install.bat)
├── models/           # Кэш HuggingFace (создаётся при запуске)
├── install.bat       # Установщик
├── run.bat           # Лаунчер (один терминал)
├── run-dev.bat       # Режим разработки (3 терминала)
├── AGENTS.md         # Гайд для разработчиков
├── update.bat        # Обновление
└── download_model.bat # Загрузчик моделей
```

## Обновление

```
update.bat
```

Обновляет ACE-Step Studio и ядро ACE-Step 1.5, затем обновляет зависимости.
